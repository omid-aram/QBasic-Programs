                     ******Mine Sweeper******

Notice:
To start the game press any key.
To select one of the icons,press TAB.
Use arrow keys to move the pointer.
You can uncover a square by pressing ENTER.
If you uncover a mine,you lose the game.
To mark a square that you suspect contains a mine,press SPACE.
You can use some shortcut keys:
  F2     ... New game
  Ctrl+x ... Exit game
  Alt+g  ... Open [game option]
When open [Best times...] or [Custom] you can close by pressing ESC.
Warning!
When you open the [Best Times...] ,if it is empty or you want to reset scores, type -omid- like this OmId.
Thank you.
